## main 함수

기본적으로 main 함수가 있으면 실행이 가능하다.

```
void main() {
    print('hello world!');
}
```

![image-20240901235624620](assets/image-20240901235624620.png)

그러나 main 함수가 아닌 다른 명칭을 가진 파일을 실행했을때 에러가 발생

```
void test() {
    print('hello world!');
}
```
